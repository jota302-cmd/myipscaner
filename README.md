# Digital Trace Auditor 🌐

**Digital Trace Auditor** (MyIPScaner) es un auditor de rastro digital diseñado para identificar tu IP, ubicación y privacidad de ISP en un solo clic, sin comprometer tu historial.

## 🚀 Características

- **Auditoría Instantánea**: Revela tu IP pública, geolocalización, ASN e información del proveedor de internet.
- **No Tracking Policy**: No almacenamos logs de tus consultas ni rastreamos tu conexión.
- **Lightning Fast**: Carga monolítica en `< 1s` utilizando el protocolo Privacy-First.
- **100% Client-Side Visualization**: Consulta APIs públicas y renderiza en tu cliente de manera segura.

## 🛠️ Especificaciones Técnicas

- **Stack**: HTML5, Tailwind CSS, Vanilla JavaScript.
- **Diseño**: Dark Luxury Identity con tipografía Inter.
- **Despliegue**: Subdominio Vercel (`myip.optimicrotools.com`).
